# Sistema de Solicitação de Compras (C#)
Basta entrar na pasta `csharp-compras` e rodar:

```
dotnet run
```

O sistema lê o arquivo `solicitacoes.txt` (um por linha: Nome;Item;Quantidade;Prioridade), mostra as solicitações e permite ordenar por item ou solicitante. Se faltar o arquivo ou alguma linha estiver errada, ele avisa na tela.

# Formulário de Serviços Internos (HTML/JS)
Abra o arquivo `js-servicos/solicitacao-servicos.html` no navegador. Preencha os campos obrigatórios e clique em Enviar. O resumo aparece na tela com um número de protocolo.

---

Projeto simples, sem dependências externas o obstaculo mais complexo foi a utilização do LINQ pois não lembrava com clareza os conceitos funcionais.

Ótimo exercício para praticar leitura de arquivos, manipulação de listas e ordenação em C#. O formulário em HTML/JS é bem básico, sem recursos externos, foi bem complexo devido estar acostumado a usar forms/fontes e 
padrões da API do google.
